# Rover Mobility Project (Rocker-Bogie vs Single-Rocker)

🚧 Status: Work in Progress

## Overview

โครงงานนี้มุ่งเน้นการออกแบบและทดสอบหุ่นยนต์เคลื่อนที่แบบ 4 ล้อ และ 6 ล้อ (Rocker-Bogie) เพื่อเปรียบเทียบประสิทธิภาพในการเคลื่อนที่บนพื้นที่ขรุขระและทางลาดชัน

## Objectives

* เปรียบเทียบความสามารถในการปีนทางลาด (15° และ 20°)
* วิเคราะห์เสถียรภาพและการกระจายน้ำหนัก
* ศึกษาประสิทธิภาพของระบบกันสะเทือน

## Current Progress

* ✅ ออกแบบและสร้างต้นแบบเสร็จแล้ว
* ✅ เริ่มทดสอบภาคสนาม
* ⚠️ อยู่ระหว่างเก็บและวิเคราะห์ข้อมูลเพิ่มเติม

## Limitations

* ข้อมูลบางส่วนยังไม่ครบ
* การทดสอบยังไม่ครอบคลุมทุกสภาพพื้นผิว

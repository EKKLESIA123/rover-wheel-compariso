## 📌 โปรเจกต์นี้ทำอะไร?

### ความเป็นมา
เราต้องการรู้ว่า หุ่นยนต์สำรวจ **4 ล้อ** กับ **6 ล้อ** ตัวไหนดีกว่า? จึงสร้างหุ่นยนต์ 2 ตัว แล้วนำไปทดลองบนพื้นเอียง เพื่อดูว่า:

- ตัวไหนเสถียรมากกว่า
- - ตัวไหนใช้พลังงานน้อยกว่า
  - - ตัวไหนมีการลื่นล้อน้อยกว่า
   
    - ### ผลที่ได้
    - | เปรียบเทียบ | 4 ล้อ | 6 ล้อ | หมายเหตุ |
    - |-----------|-------|-------|--------|
    - | ใช้พลังงาน | น้อย | มากกว่า 5-8% | 6 ล้อ หนักกว่า |
    - | เสถียรภาพ | ไม่ดี | ดีกว่า | 6 ล้อ สมดุลดีกว่า |
    - | ลื่นล้อ | มี | ไม่มี | 6 ล้อ ยึดจับพื้นได้ดี |
   
    - ### สรุป
    - - 👍 **6 ล้อ** ดีถ้าต้องการเสถียร (ปลอดภัยมากขึ้น)
      - - 👍 **4 ล้อ** ดีถ้าต้องการประหยัดพลังงาน

        ## 🔧 อุปกรณ์ที่ใช้

        ### หลัก
        - Arduino หรือ STM32 (เครื่องควบคุม)
        - - Motor DC (มอเตอร์)
          - - ล้อหุ่นยนต์
            - - ACS712 (วัดกระแสไฟ)
              - - MPU6050 (วัดการเอียง)
                - - SD Card Module (บันทึกข้อมูล)
                  - - แบตเตอรี่
                   
                    - รายละเอียดเพิ่มเติม → ดู [HARDWARE.md](./HARDWARE.md)

                    ## 🚀 วิธีเริ่มต้น

                    ### ขั้นตอนที่ 1: ติดตั้ง Python libraries

                    ```bash
                    # Copy ไปยังเครื่องของคุณ
                    git clone https://github.com/EKKLESIA123/rover-wheel-compariso.git
                    cd rover-wheel-compariso

                    # ติดตั้ง libraries ที่ต้องใช้
                    pip install -r requirements.txt
                    ```

                    ### ขั้นตอนที่ 2: เตรียมหุ่นยนต์

                    1. ประกอบหุ่นยนต์ 2 ตัว (4 ล้อ และ 6 ล้อ)
                    2. 2. ดาวน์โหลด firmware จาก `src/firmware/`
                       3. 3. อัพโหลดไปยัง Arduino/STM32
                          4. 4. ปรับแต่ง sensors
                            
                             5. ### ขั้นตอนที่ 3: ทำการทดลอง
                            
                             6. ```bash
                                # วางหุ่นยนต์บนทางเอียง 15° หรือ 20°
                                # กด start บน Arduino
                                # รอให้ข้อมูลบันทึกลงไฟล์ CSV
                                # ดึงไฟล์ CSV ออกมา
                                ```

                                ### ขั้นตอนที่ 4: วิเคราะห์ข้อมูล

                                ```bash
                                # ใส่ไฟล์ CSV ลงใน data/raw_data/
                                python src/analysis/data_analysis.py

                                # สร้างกราฟ
                                python src/analysis/plot_results.py
                                ```
